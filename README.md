# React App — Полнофункциональное приложение с маршрутизацией и API

## Описание

Учебное React-приложение, демонстрирующее:
- Маршрутизацию с React Router v6
- Работу с внешним API (JSONPlaceholder)
- Управление состоянием через Context API
- Добавление постов в избранное
- Адаптивный дизайн


## Структура проекта
src/
├── api/              # API слой
├── components/       # Переиспользуемые компоненты
├── context/          # Context API (глобальное состояние)
├── pages/            # Страницы приложения
├── App.jsx           # Корневой компонент
├── main.jsx          # Точка входа
└── index.css         # Глобальные стили
